# 反模式警报器

从巴菲特 49 个投资概念的"常见误区"中提炼的审查清单。核心知识库：`concepts/` 目录下全部 49 个概念文件，每个文件均包含"常见误区"章节。

需要查看某条反模式的完整论述和巴菲特原话时，读取对应的概念文件。

## 使用方式

当用户提出投资观点或商业分析时，对照以下 7 类反模式逐一扫描，触发的条目标注警报。

## 反模式清单

### 类别 1: 价值判断谬误

**AP-101: 把高利润等同于有护城河**
利润是结果，护城河是原因。很多企业在某个时期利润丰厚但缺乏结构性壁垒。
→ 来源：[护城河](concepts/护城河.md)

**AP-102: 把技术领先等同于持久优势**
技术优势往往是暂时的。"一条需要不断重建的护城河，最终等于没有护城河。"
→ 来源：[护城河](concepts/护城河.md)

**AP-103: 把账面价值等同于内在价值**
账面价值是历史成本的累积，内在价值是未来现金流折现。两者可能天差地别。
→ 来源：[内在价值](concepts/内在价值.md)

**AP-104: 追求精确的内在价值计算**
内在价值是一个范围而非精确数字。"大致正确好过精确错误。"
→ 来源：[内在价值](concepts/内在价值.md)

**AP-105: 把低市盈率等同于安全边际**
安全边际是价格与真实内在价值的差距，不是简单的比率指标。可口可乐在 1988 年市盈率不低，但安全边际充裕。
→ 来源：[安全边际](concepts/安全边际.md)、[买入价格](concepts/买入价格.md)

### 类别 2: 成长陷阱

**AP-201: 认为成长一定增加价值**
只有当再投资回报率 > 资本成本时，成长才增加内在价值。航空业越发展，投资者亏越惨。
→ 来源：[内在价值](concepts/内在价值.md)

**AP-202: 用绝对利润增长评估管理层**
"靠投入更多资本来赚更多钱算不上什么了不起的管理成就。你坐在摇椅上也能做到——把存款翻两番，利息也会翻两番。"
→ 来源：[留存收益](concepts/留存收益.md)

**AP-203: 为无利增长买单**
企业大量烧钱扩张但回报率低于资本成本，每一轮融资都在稀释股东价值。
→ 来源：[资本配置](concepts/资本配置.md)

### 类别 3: 管理层评估盲区

**AP-301: 管理层万能论**
"当一个声名卓著的管理层遇上一个声名狼藉的行业，最后屹立不倒的是行业的名声。"
→ 来源：[管理层](concepts/管理层.md)

**AP-302: 将个人魅力等同于管理能力**
管理层可能社交魅力十足，但在制度迫力裹挟下做出非理性决策。看行为（资本配置记录），不看人设。
→ 来源：[管理层](concepts/管理层.md)

**AP-303: 忽视激励机制**
经理人行为由激励机制塑造。过度关注短期指标的激励体系必然扭曲长期决策。
→ 来源：[管理层](concepts/管理层.md)

**AP-304: "20 年经验"幻觉**
"不是 20 年的从业经验，而是 1 年的经验重复了 20 遍。"
→ 来源：[管理层](concepts/管理层.md)

### 类别 4: 能力圈越界

**AP-401: 因为别人赚钱而越界**
"别人在自己不懂的领域赚钱，不构成自己偏离能力圈的理由。" 1999 年互联网泡沫是最生动的案例。
→ 来源：[能力圈](concepts/能力圈.md)

**AP-402: 在能力圈内犹豫不决**
错过能力圈外的机会不是错误，但在能力圈内"吮手指"才是真正的错误。
→ 来源：[能力圈](concepts/能力圈.md)

**AP-403: 将"了解产品"等同于"理解商业模式"**
每天喝可乐不等于理解可口可乐的商业经济学。能力圈要求对企业的经济特性有深层理解。
→ 来源：[能力圈](concepts/能力圈.md)

### 类别 5: 市场行为偏差

**AP-501: 用市场先生的情绪做决策**
市场先生是你的仆人，不是你的老师。价格波动是机会，不是信号。
→ 来源：[市场先生](concepts/市场先生.md)

**AP-502: 过度分散**
分散投资是"对无知的保护"。如果你真的理解一家企业，为什么只买一点点？
→ 来源：[集中投资](concepts/集中投资.md)、[分散投资](concepts/分散投资.md)

**AP-503: 试图预测市场走势**
"我们从来没有根据宏观或政治环境来决定买入或卖出。"
→ 来源：[市场预测](concepts/市场预测.md)

**AP-504: 把波动等同于风险**
真正的风险是永久性资本损失，不是短期账面浮亏。
→ 来源：[安全边际](concepts/安全边际.md)

### 类别 6: 财务分析陷阱

**AP-601: 忽视股票期权成本**
期权是真实的薪酬费用。把它排除在"调整后利润"之外是自欺欺人。
→ 来源：[管理层](concepts/管理层.md)

**AP-602: 混淆会计商誉与经济商誉**
会计商誉是收购溢价的残留物，经济商誉是企业超额盈利能力的体现。前者可能是过度支付的证据。
→ 来源：[商誉](concepts/商誉.md)

**AP-603: 用 EBITDA 衡量盈利能力**
折旧是真实的经济成本。"难道管理层以为固定资产是由牙仙子赐予的吗？"
→ 来源：[资本配置](concepts/资本配置.md)

**AP-604: 忽视保险浮存金的成本**
浮存金只有在成本低于无浮存金状况下的替代成本时才有价值。负成本浮存金才是真正的优势。
→ 来源：[保险浮存金](concepts/保险浮存金.md)

### 类别 7: 行动/决策偏差

**AP-701: 制度迫力**
管理层倾向于模仿同行、用尽预算、为交易找理由。"正派、聪明且经验丰富的管理者自然而然会做出理性的商业决策"是幻觉。
→ 来源：[管理层](concepts/管理层.md)

**AP-702: 沉没成本偏见**
继续往失败项目投钱只因为"已经投了这么多了"。巴菲特关闭纺织业务就是克服沉没成本的典范。
→ 来源：[纺织业务](concepts/纺织业务.md)

**AP-703: 杠杆迷信**
"只有在退潮的时候，你才知道谁在裸泳。" 杠杆放大收益也放大损失，且损失是不可逆的。
→ 来源：[杠杆](concepts/杠杆.md)

**AP-704: 锚定效应**
用历史买入价格或历史高点作为决策依据。价格本身不包含关于未来价值的信息。
→ 来源：[买入价格](concepts/买入价格.md)

## 审查输出模板

```markdown
# 反模式审查报告

## 触发的警报

### [AP-XXX: 名称] ⚠️
- 触发原因：[在用户分析中发现的具体表述]
- 巴菲特怎么看：[一句话总结]
- 建议：[如何修正这个认知]

### [AP-XXX: 名称] ⚠️
...

## 未触发但值得注意的
[接近触发阈值的条目，提醒用户注意]

## 总结
- 触发警报数：[X] / 25
- 风险等级：[低/中/高]
- 核心建议：[一段话]
```

## 使用原则

1. **不做道德审判**：指出认知偏差，不批评用户智商
2. **给出建设性建议**：每个警报都附带修正方向
3. **承认灰色地带**：很多情况并非非黑即白，标注不确定性
4. **鼓励独立思考**：巴菲特的框架是参考，不是教条。"即使是巴菲特自己也承认犯过很多错误"
